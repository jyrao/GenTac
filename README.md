# GenTac Project Page

Static project page for **GenTac: Generative Modeling and Forecasting of Soccer Tactics**.

## Local Preview

```bash
python3 -m http.server 8000
```

Then open:

```text
http://127.0.0.1:8000/
```

This folder is ready to be published from the `web` branch of `https://github.com/jyrao/GenTac`.
